# Mathematical Foundations

This document derives every formula that lives in `@kinemind/core-math`
and `origami-lab`. All references match the file tree of those packages
so you can audit the code line-by-line against the proofs.

## 1. The strip and its state space

A 1D origami strip is a path graph $P_N = (V, E)$ with

$$V = \{c_1, \ldots, c_N\}, \quad E = \{h_1, \ldots, h_{N-1}\}, \quad h_i = (c_i, c_{i+1}).$$

Each hinge carries a dihedral angle $\theta_i \in [-\pi, \pi]$. The
state vector is $\boldsymbol{\theta} = (\theta_1, \ldots, \theta_{N-1})^\top \in \mathbb{R}^{N-1}$.

Mountain / valley assignments fall out of the sign of $\theta_i$:

$$a_i = M \iff \theta_i > 0, \quad a_i = V \iff \theta_i < 0, \quad a_i = F \iff \theta_i = 0.$$

## 2. SE(3) forward kinematics

Cell 0's local frame coincides with the world frame. Cell $i$'s frame is
obtained from cell $i-1$'s frame by

1. translating along the cell's local +X axis by the cell length $L_{i-1}$,
2. rotating about the cell's local +Y axis by the hinge angle $\theta_{i-1}$.

Let $T_i = \mathrm{Trans}(L_i \hat{\mathbf{e}}_x) \cdot \mathrm{Rot}(\hat{\mathbf{e}}_y, \theta_i) \in SE(3)$.
Then the cumulative pose of cell $i$ is

$$M_i = T_1 \cdot T_2 \cdots T_{i-1} \in SE(3), \qquad M_0 = I.$$

The leading-edge midpoint of cell $i$ is $\mathbf{p}_i = M_i \cdot \mathbf{0}$.

In the implementation this is one O(N) loop over `forwardKinematics` /
`forward_kinematics`; quaternions are used for rotation accumulation to
avoid gimbal lock and limit drift.

## 3. The V₄ symmetry group

A 1D strip is invariant under the Klein four-group $V_4 = \mathbb{Z}_2 \times \mathbb{Z}_2$
generated by

* $\sigma$ (reflection through the strip midpoint, $h_i \mapsto h_{N-i}$),
* $\tau$ (paper-flip, $\theta_i \mapsto -\theta_i$).

Note: full dihedral groups $D_n$ act on closed cyclic graphs; an open path
admits only $V_4$. The action on the state vector is

$$P_\sigma = \begin{pmatrix} 0 & \cdots & 0 & 1 \\ \vdots & & 1 & 0 \\ 0 & 1 & & \vdots \\ 1 & 0 & \cdots & 0 \end{pmatrix} \quad (\text{anti-diagonal permutation matrix}), \qquad P_\tau = -I.$$

`packages/core-math/src/symmetry.ts::kleinFourStrip` constructs both.

## 4. The mental coupling matrix

We define $C \in \mathbb{R}^{(N-1) \times (N-1)}$ by

$$C_{ij} = \left.\frac{\partial \theta_j^{\text{mental}}}{\partial \theta_i^{\text{intent}}}\right|_{\boldsymbol{\theta}=0}.$$

KMM hypothesis: cognition respects $V_4$ symmetry, so $C$ is
$V_4$-equivariant — i.e. $P_g C P_g^{-1} = C$ for all $g \in V_4$.

### 4.1 Schur-lemma reduction

Decompose $\mathbb{R}^{N-1}$ into $\sigma$-eigenspaces:

$$V_+^\sigma = \{\boldsymbol{\theta} : P_\sigma \boldsymbol{\theta} = \boldsymbol{\theta}\}, \qquad V_-^\sigma = \{\boldsymbol{\theta} : P_\sigma \boldsymbol{\theta} = -\boldsymbol{\theta}\}.$$

For odd $N-1$ these have dimensions $\lceil (N-1)/2 \rceil$ and
$\lfloor (N-1)/2 \rfloor$. By Schur's lemma any $V_4$-equivariant
operator decomposes as

$$C = c_+ \Pi_{V_+^\sigma} + c_- \Pi_{V_-^\sigma},$$

so the parameter family collapses from $(N-1)^2$ free entries to **two**
scalars.

### 4.2 Mirror coupling matrix

The simplest non-trivial point is

$$C(\beta)_{ij} = \begin{cases} 1 & i = j \\ \beta & j = N-1-i, \, i \ne N-1-i \\ 0 & \text{otherwise.} \end{cases}$$

Eigenvalues are $\{1+\beta, 1-\beta\}$ each with multiplicity
$\lfloor (N-1)/2 \rfloor$, plus a single $1$ when $N-1$ is odd.

`mirrorCouplingMatrix(n, beta)` builds this; tests verify equivariance,
spectral radius, and effective rank.

### 4.3 Estimation from data

Given $K$ trials $(\boldsymbol{x}^{(k)}, \boldsymbol{y}^{(k)})$ stack them into matrices
$X, Y \in \mathbb{R}^{K \times (N-1)}$. The Tikhonov-regularised LSQ estimator is

$$\hat{C}^\top = (X^\top X + \lambda I)^{-1} X^\top Y.$$

When a symmetry group $G$ is supplied, project to the equivariant
subspace via the Reynolds operator

$$\hat{C}^G = \frac{1}{|G|} \sum_{g \in G} P_g \, \hat{C} \, P_g^{-1}.$$

`estimateCoupling(intents, responses, { lambda, group })` performs both
steps; the projection is idempotent and is verified in
`tests/coupling.test.ts`.

## 5. Effective rank and spectral radius

We use power iteration for the spectral radius and Jacobi eigen
decomposition of $M M^\top$ for effective rank, both audited and
dependency-free; small $N \le 50$ keeps wall time below a millisecond.

## 6. Self-intersection (broad phase)

Cells are unit squares attached at hinge edges. World-frame corners are
computed in O(N), AABBs in O(N), and an O(N²) AABB-overlap pass with
the i / i+1 adjacency exclusion locates colliding pairs. The narrow-phase
SAT check is shipped behind a follow-up release; the broad pass is
already sufficient for all interactive viewer use cases.
